EDM Council FIBO in the UML™ Concept Modeling Profile

The Unified Modeling Language (UML™) Concept Modeling Profile (UCMP) displays FIBO as hyperlinked class diagrams in a UML™ model.

  • The FIBO production model is available for the Cameo Concept Modeler (CCM), which can be freely downloaded for evaluation purposes.
  • A quick-reference poster explains the notation used in this model
  • The specification for this UML™ profile is currently undergoing standardization by No Magic, Inc., Model Driven Systems, Inc., and the EDMCouncil.
    • The profile in .mdzip format is available for MagicDraw.
    • The profile in XMI™ is available for all other UML™ standards-compliant tools.
