Products

FIBO is being developed in a continuous process by FIBO Content Teams (FCT) according to the FIBO Build-Test-Deploy-Maintain Methodology. All of FIBO is continuously published with the maturity level indicated as Development. This is all of FIBO. In addition, the parts of FIBO that have been substantially vetted and tested are published at the end of each calendar quarter in a maturity level indicated as Production. All other content, including parts of these domains in published FIBO Development, are for information only, and should not be relied upon except as raw material.

FIBO Products Today, FIBO is available as the products below, with more soon to follow. All FIBO products are generated or derived from the W3C OWL System of Record. Be aware that flaws in the OWL will be reflected in all other products. We are making all of FIBO available now because we have already seen amazing work done with FIBO and we welcome, almost demand, your feedback. dwisnosky@edmcouncil.org.

fibopedia Annotated structure of FIBO ontologies and modules. Includes web navigation as well as a machine-readable module structure (in RDF). FIBOPedia provides access to ontology documentation generated using WIzard for DOCumenting Ontologies (WIDOCO) software, which includes a graphical visualization using Visual Notation for OWL Ontologies (VOWL).
glossary Web searchable, downloadable in CSV and/or Excel Includes web-searchable people and machine generated definitions, and as a spreadsheet that may be downloaded and worked with offline
data dictionary data dictionary searchable from the Web and down loadable in csv or .xls A summary of each operational class in FIBO, with its properties, presented in the form of tables and fields for viewers familiar with relational data dictionaries.
vocabulary The FIBO Vocabulary in SKOS FIBO rendered using the W3C SKOS standard, using extensions to represent FIBO relationships.
smif Sample UML Diagrams of FIBO. The Semantic Modeling for Information Federation (SMIF) specification allows round tripping between UML and OWL
ontology The FIBO Ontologies These are the ontologies for the FIBO Domains developed by the FIBO Team for nearly a decade. They are in W3C OWL and can be opened with any RDF-compliant tool. FIBO is published in multiple RDF serializations, including RDF/XML, Turtle, JSON-LD and N-Quads. See the Supported Formats table below to access FIBO in these forms. To see individual FIBO ontology files, please see the FIBO Ontology File Directory
linked data fragments The FIBO Linked Data Fragments This provides a dynamic query interface to the FIBO content using Linked Data Fragments.
UMLCMP UML Concept Modeling Profile The Unified Modeling Language (UML) Concept Modeling Profile (UCMP) displays FIBO as hyperlinked class diagrams in a UML model.
schema.org The FIBO schema.org files This page provides background information on the use of schema.org for marking up banks and their products.

I want to contribute!

FIBO is a community effort - we want to get many well-informed people from the Finance Industry involved. This includes subject matter experts, architects, ontologists and even the "just curious". In the style of collaborative, open source projects, FIBO collaboration is managed through GitHub, Jenkins and by our use of Atlassian JIRA and Confluence. The FIBO Team has developed many tools and practices to help your contributions work well with FIBO. For information on how to get them, please see the Developers' Guide.
